# Task Management Frontend

## Short Description
This is the frontend for the Task Management application, built using React, React Router, and DaisyUI. The app allows users to create, view, edit, delete, and categorize tasks. It interacts with the backend via RESTful APIs and incorporates JWT authentication. Real-time updates are supported through TanStack (React Query) for a fast and seamless experience. The application is responsive and uses Tailwind CSS for styling.

## Dependencies
- `react`: JavaScript library for building user interfaces.
- `react-router-dom`: For routing and navigation.
- `axios`: For making HTTP requests to the backend API.
- `react-query`: For data fetching, caching, and synchronization.
- `daisyui`: UI component library based on Tailwind CSS for modern design.
- `tailwindcss`: A utility-first CSS framework.
- `react-icons`: A library for customizable icons.
